# POS 收銀管理系統 — 規格書 v1.0

> OpenClaw 一人公司 SaaS 系列
> 從 Notion「POS 系統（行業切換）」v2.2.1 啟發，本 MVP 重新設計為通用收銀版
> 最後更新：2026-07-13

---

## 一、產品定位

### 目標使用者
- 台灣微型店家（小吃店、咖啡廳、零售小店）
- 連鎖加盟主（多店管理）
- 想跨業態試水的創業者
- 個人工作室 / 接案者

### 核心價值
- **零月費**：純前端 SPA，所有資料存於 localStorage
- **即開即用**：不用註冊、不用綁定信用卡、不用裝 App
- **8 大模組**：從收銀到報表一站搞定
- **可離線**：完全在瀏覽器運作，網路斷了照常營業

### 與「POS 系統（行業切換）」差異
- 原版：3 行業模板（餐飲/零售/服務）切換
- 本版：**通用版**，所有行業都能用，模組化設計

## 三、8 大模組

### 模組 1：收銀介面（Register）
- 商品選擇（點擊 / 分類篩選 / 關鍵字搜尋）
- 條碼掃描（鍵盤輸入模擬）
- 購物車（增/減/刪/清空）
- 會員選擇（累積點數）
- 4 種付款方式：現金 / 信用卡 / Line Pay / 街口
- 現金找零計算
- 折扣：0% / 5% / 10% / 20%
- 交易完成確認（顯示訂單編號 + 找零）

### 模組 2：庫存管理（Inventory）
- 商品 CRUD（新增 / 編輯 / 刪除 / 啟用切換）
- 欄位：SKU、名稱、分類、售價、成本、庫存、安全庫存、單位
- 低庫存自動警示（紅色 badge）
- 毛利即時計算 = ((售價-成本)/售價) × 100%
- 分類篩選 + 關鍵字搜尋
- 預載 demo：20 項商品（飲料/烘焙/輕食/零食/日用品）

### 模組 3：會員 CRM（Members）
- 會員 CRUD
- 4 級會員：Bronze / Silver / Gold / VIP
  - 升級規則：消費 ≥ 5,000 → Silver；≥ 20,000 → Gold；≥ 50,000 → VIP
- 點數累積（每消費 10 元累積 1 點，可設定）
- 儲值卡餘額
- 儲值功能
- 會員消費歷史

### 模組 4：交易紀錄（Transactions）
- 交易列表（訂單編號 / 時間 / 會員 / 收銀員 / 付款 / 金額）
- 篩選：狀態（已完成/已退款/已作廢）+ 關鍵字
- 統計：總營收、平均客單
- 詳情查看 + 退款 / 作廢功能（需填原因）
- 作廢/退款會自動還原庫存

### 模組 5：報表中心（Analytics）
- 時間範圍：今日 / 本週 / 本月 / 全部
- 核心指標：營收 / 訂單數 / 平均客單 / 活躍會員
- 毛利分析（含毛利率進度條）
- 最近 14 天營收趨勢圖（bar chart）
- 熱銷商品 TOP 5（emoji 金銀銅）
- 會員消費排行 TOP 5
- 付款方式分佈（現金/信用卡/Line Pay/街口）

### 模組 6：員工管理（Staff）
- 員工 CRUD
- 3 種角色：管理員 / 店長 / 收銀員
- 上下班打卡
- 員工業績排行（交易數 + 總營收）

### 模組 7：AI 經營助理（AI Assistant）
- 今日重點建議（高峰時段 + 客單分析）
- 低毛利商品警示（毛利 < 40%）
- 即將缺貨預警
- 銷售下滑商品偵測（本週 vs 上週，衰退 30%+）
- 銷售成長商品（本週 vs 上週，成長 30%+）
- 高貢獻會員識別（消費 > 平均 2 倍）

### 模組 8：系統設定（Settings）
- 店家資訊（名稱/地址/電話）
- 稅務設定（稅率 / 內含/外加）
- 會員點數規則
- 收據頁尾訊息
- 重置 Demo 資料（危險操作）
